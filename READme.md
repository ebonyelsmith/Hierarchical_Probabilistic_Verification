# From Global to Local: Hierarchical Probabilistic Verification for Reachability Learning
### [Paper](https://arxiv.org/)<br>

Ebonye Smith, Sampada Deglurkar, Jingqi Li, Gechen Qu, Claire Tomlin<br>
University of California, Berkeley<br> 
University of Texas at Austin

This repository contains the implementation of a **Hierarchical Safe, Target-Reaching Framework** that combines global probabilistic verification with online local refinement for an autonomous drone racing case study. We leverage scenario optimization theory to provide probabilistic safety guaranees for globally and locally verified sets.
